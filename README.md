# Laboratorio Programación C++

Este repositorio contiene la solución a los siguientes ejercicios:

## 1. LeetCode 171 – Excel Sheet Column Number
Conversión de columna de Excel a número (base 26).
Complejidad: O(n)

## 2. LeetCode 121 – Best Time to Buy and Sell Stock
Cálculo de máxima ganancia con recorrido único.
Complejidad: O(n)

## 3. Collatz Conjecture – Exercism
Cálculo de pasos hasta llegar a 1 con validación de entrada.

## 4. Programa con Callbacks
Programa que recibe argumentos por consola:
./operacion <num1> <operador> <num2>

Implementa:
- Suma
- Resta
- Multiplicación
- División
